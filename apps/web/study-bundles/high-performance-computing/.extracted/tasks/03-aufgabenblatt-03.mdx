## High Performance Computing (CDS-110)
Aufgabenblatt 3: Netztopologien I

## Aufgabe 1
Die Evaluation von Netztopologien basiert auf unterschiedlichen Aspekten wie Durchmesser,
Bisektionsweite oder Kosten (d.h. Anzahl der Kanten).
a) Geben Sie eine Formel nur in Abhängigkeit von N (Anzahl der Knoten) und d (Dimension der
Netztopologie) an, mithilfe derer sich der Durchmesser und die Bisektionsweite folgender
Netztopologien berechnen lässt.
      - Lineares Array
      - Ring
      - Binärbaum
      - d-dimensionaler Torus
      - d-dimensional Hyperwürfel
b) Bestimmen Sie die Kosten (Anzahl der Kanten) obiger Netztopologien nur in Abhängigkeit von
N und d.
c) Vergleichen Sie obige Netztopologien gemäss den gegebenen Aspekten und erklären Sie,
welche Topologie Ihrer Meinung nach das beste Kosten-Nutzen-Verhältnis aufweist.

## Aufgabe 2
a) Gegeben sei ein 3-dimensionaler Torus der Grösse 8 x 8 x 16 (d.h. 1024 Koten).
I. Berechnen Sie die Bisektionsweite dieser Netztopologie.
II. Berechnen Sie den Durchmesser dieser Netztopologie.
III. Berechnen Sie die Kosten (d.h. Anzahl der Kanten) dieser Netztopologie.
b) Gegeben sei ein 10-dimensionaler Hyperwürfel (d.h. 1024 Knoten).
I. Berechnen Sie die Bisektionsweite dieser Netztopologie.
II. Berechnen Sie den Durchmesser dieser Netztopologie.
III. Berechnen Sie die Kosten (d.h. Anzahl der Kanten) dieser Netztopologie.
c) Entscheiden Sie in Abhängigkeit obiger Ergebnisse, welche Netztopologie Sie aus einer Kosten-
Nutzen-Perspektive zur Vernetzung von 1024 Knoten wählen würden. Begründen Sie Ihre
Antwort.

Die Bearbeitung der Aufgaben erfolgt freiwillig.
